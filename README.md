# Como executar o projeto

## Mobile
Pré-requisitos: Expo

```bash
# clonar repositório
git clone https://github.com/Beluhga/GoFinances.git

# entrar na pasta do projeto Mobile
cd GoFinances

# instalar dependências
yarn install

# .eng
Realocar o seu proprio cliente com o do google

# executar o projeto
expo start

```
